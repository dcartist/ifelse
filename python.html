Let's just say you have a light switch set
to off


						light = "off"

You tell someone to turn the light on it if the light is off it
would look like this:


						light = "off"
						if light == "off":
							light = "on"

Now you can tell them if it's off turn it on
or if it's on turn it off:


						light = "on"
						if light == "off":
							light = "on"
						else:
							light = "off"
